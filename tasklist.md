# PlayHub 개발 Tasklist

## 📜 개발 규칙
1.  **Task 단위 개발**: 아래 정의된 Task를 순서대로 하나씩 완료합니다.
2.  **Playwright 테스트**: 각 Task의 기능 구현이 끝나면, 해당 기능이 올바르게 동작하는지 검증하는 Playwright 테스트 코드를 작성하고 실행합니다.
3.  **완료 체크**: 테스트가 성공적으로 통과하면, 해당 Task의 체크박스를 `[x]`로 표시하여 완료 처리합니다.
4.  **다음 Task 진행**: 완료 처리 후, 다음 Task 개발을 시작합니다.
